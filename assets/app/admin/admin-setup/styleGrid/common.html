Components
  • Action Buttons
    Set has-type-permission | has-type-module as per permissions.
    Set expression type as follows:
    *-exp-type="'fn'" // For functions
    *-exp-type="'sref'" // For new state
    {{vm.actionBtnComponent}}
  • Excel Import/Export Dropdown
    Set grid (optional) value to 'true' to export specific columns from the collection; 'false' for all columns.
    Set callback (optional) function for future processing of data.
    {{vm.excelBtnComponent}}
  • Content Loader
    Set message (optional) to set custom message. Default is 'No Records Found!'
    {{vm.loaderComponent}}
  • URL Binder
    {{vm.urlProtocolComponent}}
  • Copy to Clipboard Button
    Set vm.copyToClipboardFn = CommonService.copyToClipboardFn; this line in your controller.
    vm.copyToClipboardFn(null, 'pass data to be copied')
    {{vm.copyBtnComponent}}
  • File Upload
    Set btn-text value to your desired name. Default is '{{vm.fuTypeArr[vm.fuType - 1].defaultBtnText}}'
    Set folder value to your given folder name.
    {{vm.fuTypeArr[vm.fuType - 1].content}}
  • Pagination
    {{vm.paginationComponent}}
